============
Installation
============


Install ohno from PyPi:

.. code-block:: console

    $ pip install ohno


or development from the code:

.. code-block:: console

    $ git clone https://github.com/vsoch/ohno
    $ cd ohno
    $ pip install -e .


Next check out the :ref:`getting_started-user-guide` pages for more detail to use the library.
